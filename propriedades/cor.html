Propriedade: Cor

Propriedade: Color

Padrão de cores:

Nominal

Utiliza o nome em inglês.

Exemplo de código:

h1 { color: blue; }

Hexadecimal

Utiliza caracteres hexadecimais.

Exemplo de código:

h1 { color: #0000FF; }

RGB

Utiliza a combinação de valores de 0 a 255.

Exemplo de código:

h1 { color: rgb (0, 255, 255); }

Observação:

Sempre que quiser alterar a cor de texto utilizar a propriedade color.

Propriedade: Background-Color

Utilizada para dar cor para o fundo de determinado bloco da página.

Exemplo de código:

p { background-color: gray; }
