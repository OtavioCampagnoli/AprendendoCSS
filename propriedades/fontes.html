Fontes

Propriedade: font-family

Propriedade que define a família da fonte, cada família tem um estilo diferente de fonte.

Fontes padrões

As fontes padrões são aquelas que já vem por padrão no navegador.

Por exemplo:

  • Verdana
  • Brush Script mt
  • Comic Sans MS
  • ORC A
  • Courier
  • Arial
  • Helvetica
  • Stencil
  • Impact
  • Arial Narrow

Exemplo de código:

h2 { font-family: 'Verdana'; }

Propriedade: font-weight

Espessura da fonte

Com essa propriedade podemos mudar a espessura da fonte, recebendo como parâmetro normal ou bold, deixando mais 'fina' ou mais 'grossa'.

Exemplo de código:

h1 { font-weight: bold; }

Mundando espessura através da tag: span

Utilizamos essa tag, caso precise marcar uma parte do texto, por exemplo abrimos a tag span e depois fechamos para marcar determinado texto. span

E quando precisarmos destacar determinado texto, podemos apenas atribuir à tag span no CSS, usando a propriedade font-weight.

Exemplo de código:

span { font-weight: bold; }

Propriedade: text-decoration

Mudando decoração do texto com: text-decoration

Essa propriedade nos permite colocarmos um sublinhado (overline) sobre, embaixo (underline) e nenhum (none), caso já tenha algo sublinhado.

Exemplo de código:

span { text-decoration: underline; }

Propriedade: text-align

Mudando alinhamento dos elementos com: text-decoration

Essa propriedade nos permite alinhar os elementos centralizando (center), à esquerda (left), à direita (right) e justificado (justify).

Exemplo de código:

h1 { text-align: center; }

Propriedade: line-height

Mudando alinhamento dos elementos com: text-decoration

Essa propriedade nos ajustar a altura da linha de determinado elemento.

Exemplo de código:

h1 { text-align: line-height: 48px; }

Observação:

Ao utilizar as propriedades font-size e line-height inserimos um número seguido por px. Pixel (px) é uma unidade de medida. Existem outras como, por exemplo, cm (centímetros), mm (milímetros), in (polegadas), entre outras, porém, pixel (px) é a mais utilizada nos códigos CSS.
