Fontes

Propriedade: font-family

Propriedade que define a família da fonte, cada família tem um estilo diferente de fonte.

Fontes padrões

As fontes padrões são aquelas que já vem por padrão no navegador.

Por exemplo:

  • Verdana
  • Brush Script mt
  • Comic Sans MS
  • ORC A
  • Courier
  • Arial
  • Helvetica
  • Stencil
  • Impact
  • Arial Narrow

Exemplo de código:

h2 { font-family: 'Verdana'; }

Fontes externas

Utiliza caracteres hexadecimais.

Exemplo de código:

h1 { color: #0000FF; }

RGB

Utiliza a combinação de valores de 0 a 255.

Exemplo de código:

h1 { color: rgb (0, 255, 255); }

Observação:

Sempre que quiser alterar a cor de texto utilizar a propriedade color.

Propriedade: background-color

Utilizada para dar cor para o fundo de determinado bloco da página.

Exemplo de código:

p { background-color: gray; }
